- Functions in JavaScript
 - 1.Parameters in JavaScript Functions 
   - Parameters are veriables listed in a function definition.
   - Arguments are the actual values passed to the function when it is invoked.

  - 1.Required Parameters
    - These parameters must be provided when the function is called. If not passed, their value is `undefined`.
    - Example:
      ```javascript
      function add(a, b, c) {
        console.log(c) // Output: undefined
        console.log(a + b); // Output: 5
      }
      console.log(add(2, 3, )); // Output: 9
      ```
  - 2.Default Parameters
    - These parameters have default values assigned to them. If no value is provided during the function
    call, the default value is used.
    - Example:
      ```javascript
      function multiply(a, b = 1) {
        return a * b;
      }
      console.log(multiply(5)); // Output: 5 (b defaults to 1)
      console.log(multiply(5, 2)); // Output: 10
      ```
  - 3.Rest Parameters
    - These parameters allow a function to accept an indefinite number of arguments as an array.
    - Example:
      ```javascript
      function sum(...numbers) {
        return numbers.reduce((total, num) => total + num, 0);
      }
      console.log(sum(1, 2, 3)); // Output: 6
      console.log(sum(4, 5, 6, 7, 8)); // Output: 30
      ```
 - 2.Arguments in JavaScript Functions
    - Arguments are the actual values passed to the function when it is invoked.
  - 1.Positional Arguments
    - These arguments are passed to the function in the order they are defined in the function signature.
    - Example:
      ```javascript
      function greet(name, message) {
        console.log(message + ', ' + name + '!');
      }
      greet('Alice', 'Hello'); // Output: Hello, Alice!
      ```
  - 2.Default Arguments
    - These arguments correspond to parameters that have default values. If no value is provided, the default is used.
    - Example:
      ```javascript
      function greet(name, message = 'Hello') {
        console.log(message + ', ' + name + '!');
      }
      greet('Bob'); // Output: Hello, Bob!
      greet('Bob', 'Hi'); // Output: Hi, Bob!
      ```
  - 3.Spread Arguments
    - These arguments allow an array to be expanded into individual elements when passed to a function.
    - Example:
      ```javascript
      function sum(a, b, c) {
        return a + b + c;
      }
      const numbers = [1, 2, 3];
      console.log(sum(...numbers)); // Output: 6
      ```
 - 3.Variable Hoisting in JavaScript Functions
   - Hoisting is the default behavior in JavaScript where variable declartions are moved to the top of their scope befor execution,
   However, only the declarations are hoisted, not the initializations.

   key Points:
   - Var variables are hoisted and initialized with `undefined`.
   - let and const variables are also hoisted but remain in the "temporal dead zone"(TDZ) until their declaration line executes.
   - Accessing let or const variables before their declaration results in a ReferenceError.
    - Example:
      ```javascript
      console.log(x); // Output: undefined
      var x = 5;
      console.log(x); // Output: 5
      ```
 - 4.Function Hoisting
   - Function declarations are hoisted entirely, meaning you can call the function before its declaration in the code.
   key Points:
   - Function declarations are hoisted completely, allowing calls before definition.
   - function expressions and arrow functions are not hoisted since they're treated as variable assignments.
   - Example:
     ```javascript
     greet(); // Output: Hello, World!

     function greet() {
       console.log('Hello, World!');
     }
     ```
   - Function expressions (including arrow functions) assigned to variables are not hoisted in the same way.
   - Example:
     ```javascript
     greet(); // Error: greet is not a function

     var greet = function() {
       console.log('Hello, World!');
     };
     ```
 - 5.Classic Function and nested functions
   - Classic function are used the Function keyword.
   - nested function are function inside other function.
     
    Example : 
    ```javascript
    function outerFunction(outerVariable) { // outer function
      return function innerFunction(innerVariable) { // nested function
        console.log('Outer Variable: ' + outerVariable);
        console.log('Inner Variable: ' + innerVariable);
      };
    }
    ```
 - 6.scope chain in JavaScript
    - the scope chain determines the accessibility of variables, objects, and functions in different parts of the code.
    - when a function is executed, it creates a new scope. if the function needs to access a variable, it first looks in its own scope. if not found, it looks in the outer scope, and so on, until it reaches the global scope.
    - Example:
    ```javascript
    function outerFunction() {
      var outerVariable = 'I am from outer function';

      function innerFunction() {
        var innerVariable = 'I am from inner function';
        console.log(outerVariable); // Accessing outer function variable
        console.log(innerVariable); // Accessing inner function variable
      }

      innerFunction();
      // console.log(innerVariable); // Error: innerVariable is not defined
    } 
    outerFunction();
    ```
 - 7.IFFE (Immediately Invoked Function Expression)
   - An IIFE is a JavaScript function that runs as soon as it is defined.
   - It is a design pattern which is also known as a Self-Executing Anonymous Function and contains two major parts.
     - The first is the anonymous function with lexical scope enclosed within the Grouping Operator `()`.
     - The second part creates the immediately invoked function expression `()`, through which the JavaScript engine will directly interpret the function.
   - Example:
     ```javascript
     (function() {
       console.log('This function runs immediately upon definition!');
     })();
     ```
 - 8.More type of functions in JavaScript
   - Arrow Functions
     - A shorter syntax for writing function expressions using the `=>` syntax.
     - Example:
       ```javascript
       const add = (a, b) => a + b;
       console.log(add(2, 3)); // Output: 5
       ```
   - Anonymous Functions
     - Functions without a name, often used as arguments to other functions or assigned to variables.
     - Example:
       ```javascript
       setTimeout(function() {
         console.log('This runs after 1 second');
       }, 1000);
       ```
   - Callback Functions
     - Functions passed as arguments to other functions and executed after some operation is completed.
     - Example:
       ```javascript
       function fetchData(callback) {
         setTimeout(() => {
           const data = 'Sample Data';
           callback(data);
         }, 1000);
       }

       fetchData(function(data) {
         console.log('Received:', data);
       });
       ```
   - Generator Functions
     - Functions that can be paused and resumed, defined using the `function*` syntax.
     - Example:
       ```javascript
       function* generatorFunction() {
         yield 1;
         yield 2;
         yield 3;
       }

       const gen = generatorFunction();
       console.log(gen.next().value); // Output: 1
       console.log(gen.next().value); // Output: 2
       console.log(gen.next().value); // Output: 3
    - Higher-Order Functions
      - Functions that take other functions as arguments or return functions as their result.
      - Example:
        ```javascript
        function higherOrderFunction(fn) {
          return function(x) {
            return fn(x) * 2;
          };
        }

        const double = higherOrderFunction(function(num) {
          return num + 3;
        });

        console.log(double(5)); // Output: 16
        ```
    - first Class function
         - Functions that can be treated like any other variable. They can be assigned to variables, passed as arguments, and returned from other functions.
        - Example:
            ```javascript
            function greet(name) {
            return 'Hello, ' + name + '!';
            }
    
            const sayHello = greet; // Assigning function to a variable
            console.log(sayHello('Alice')); // Output: Hello, Alice!
            ```
